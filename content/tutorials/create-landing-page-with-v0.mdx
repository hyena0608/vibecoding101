## What You'll Build

A professional-looking landing page with:
- Hero section with a headline and call-to-action
- Features section with icons
- Pricing table
- Footer

All created by describing what you want in plain English.

## Before You Start

You'll need:
- A web browser (Chrome, Safari, Firefox, etc.)
- A [v0.dev](https://v0.dev) account (free to sign up)
- 5-10 minutes

That's it. No downloads, no installation, no setup.

<Step number={1} title="Open v0 and Start a New Project">

Go to [v0.dev](https://v0.dev) and log in. Click **New Chat** to start a fresh project.

You'll see a text input where you can describe what you want to build.

</Step>

<Step number={2} title="Describe Your Landing Page">

Type a prompt like this:

```
Create a landing page for a productivity app called "FocusFlow."

Hero section: Large headline "Stay Focused. Get More Done." with a subtitle
about an AI-powered focus timer. Two buttons: "Start Free Trial" (primary)
and "Watch Demo" (outline).

Features section: 3 cards in a row with icons.
- Smart Focus Timer (clock icon)
- Distraction Blocker (shield icon)
- Progress Analytics (chart icon)

Pricing: 3 tiers - Free, Pro ($9/mo), and Team ($29/mo).

Footer with links and copyright.

Use a modern, clean design with purple as the accent color.
```

Press Enter and wait a few seconds. v0 will generate a complete landing page.

</Step>

<Step number={3} title="Preview and Refine">

v0 shows you a live preview immediately. Look at the result and ask for changes:

- *"Make the hero section taller with more padding"*
- *"Change the background to a subtle gradient"*
- *"Add a gradient background to the hero section"*
- *"Make the pricing cards have hover effects"*

Each change updates the preview instantly.

</Step>

<Step number={4} title="Add More Sections">

Want to add more? Just ask:

- *"Add a FAQ section with 5 common questions"*
- *"Add a 'How it Works' section with 3 numbered steps"*
- *"Add an email signup form above the footer"*

v0 handles layout, spacing, and responsiveness automatically.

</Step>

<Step number={5} title="Export and Deploy">

When you're happy with the result:

1. Click **Deploy** in v0 to get a live URL instantly
2. Or click **Code** to see the generated React code
3. You can also push to a GitHub repo and deploy on Vercel

Your landing page is live and ready to share!

</Step>

## Try It Yourself

Ready to build your own? Head over to [v0.dev](https://v0.dev) and paste the prompt from Step 2 above. You'll have a working landing page in under a minute.

<Callout type="tip">
Try modifying the prompt — change "FocusFlow" to your own app name and adjust the features. The best way to learn is by experimenting!
</Callout>

## Pro Tips for Better Results

1. **Be specific about layout** — "3 cards in a row" is better than "some feature cards"
2. **Mention the color scheme** — "Purple accent, light background" gives consistent results
3. **Reference real sites** — "Similar to the Stripe pricing page" helps the AI understand the vibe
4. **One section at a time** — If the initial result isn't perfect, refine section by section
5. **Use images** — You can paste a screenshot or sketch and say "Make it look like this"

## What Can You Build with v0?
